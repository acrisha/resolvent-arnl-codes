# resolvent-arnl-codes
This repository contains the resolvent mode and nonlinearity codes used in the Madrid summer 2025 workshop proceedings and used for the followup full archival manuscript (summer 2026).
This repository contains the resolvent mode code, nonlinearity code, and post processing scripts used to analyze data. 
This repository also contains an additional README to understand the scripts (.m) and the associated input files (.mat) and output files (.mat).

Large input files (i.e., XXmode.mat) are located in shared OneDrive (cannot be stored on GitHub). 
Large output files (with results/raw data) are also located in shared OneDrive folder.
