# resolvent-arnl-codes
This repository contains the resolvent mode and nonlinearity codes used in the Madrid summer 2025 workshop proceedings and used for the followup full archival manuscript (summer 2026).
This page contains the resolvent mode code, nonlinearity code, and post processing scripts used to analyze data. 
This page also contains an additional README to understand the scripts (.m) and the associated input files (.mat) and output files (.mat).
